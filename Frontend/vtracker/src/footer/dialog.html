FAQ

Hey, you're missing someone!

This website only collects data on streamers who self-report as a VTuber using the tag system on Twitch. If there is a streamer that you believe should be listed on this website who is absent, feel free to contact me with their information. Additionally, if you are a streamer yourself and for whatever reason would like to be removed from the website, let me know.

Can I request a feature/contribute?

Sure. No promises that it will get implemented, though - I work on this as a side project when I have the time for it, so it's not my primary focus.
If you have a feature request, you can send me an email with the details. If you have a direct contribution to make, check out the GitHub repository and submit a pull request.

Is there an API?

There is no API at this time; however, all of the data that is used to generate content on this website is publicly available in their most up-to-date form through the following links:
- Vtuber Twitch Data (Data such as stream status, channel tags, stream title, etc.)
- VTuber History Data (Data such as daily/monthly average viewers/stream duration, etc.)

This website is a personal project; I would request that if you wish to make use of the data it generates, grab it from the above links - please do not scrape this site.


NOTE:

This website is still in development, and as such is prone to bugs - these can range from minor UI disruptions to entire functionality blackouts as the site and backend components are being developed and updated. I hope you enjoy using this website, but please do not rely on it for anything other than personal usage.
